Tag! You're It!

Play tag around the world with your friends

What is it anyways?

Tag! You're It! is an Android game that lets you play tag with anyone else around the world. Enough of only being able to only tag as far as your arm is long, and extend your reach. If you'd like to see it in action (or listen to my beautiful voice), watch the gameplay information video shown here.

Onigokko

After typing "Tag! You're It!" a couple hundred times, I decided to change any code references to the name of the project to 'Onigokko', which is the variation of tag in Japan. Of course, the release name will remain the same. If you're looking for a more in depth explanation about the structure of the project or want to see some actually pertinent information about the development process, here is the documentation.
