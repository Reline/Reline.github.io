Project Runner GitLab

An Android application using C++ through the Android Native Development Kit, Project Runner is a 2D Top-Down, endless running platformer where the player looks down at the player and leads their character through a never ending dungeon. Design inspired by Temple Run and art inspired by The Impossible Game, the player will use swipes to change lanes and directions in order to avoid obstacles to achieve a high score.

Tag! You're It! GitHub
Nathan Reline

Tag! You're it! is an Android application that allows users to play tag but without having to physically touch one another. Players will be alerted if others in the same game are nearby. Players will then proceed to either avoid being tagged or chase their target. The distance between players will be reported to the user but not the direction they are, which keeps the players on their toes. The minimum distance at which players need to be will be customizable, as well as the number of players who can be 'it' or start out as 'it'.

Intermediary GitHub
Nathan Reline

An intermediary app for those who have trouble speaking their own or other languages. Choose any supported locale on Android to say any text you input. Only installed languages are supported. Default language is Japanese. Use case example: hearing Kanji pronounced to skip looking up furigana.

Java Function Tester GitHub
Justin Fern, Kyle Huntsman, Nathan Reline

The Java Function tester was an application created for a High School teacher to help with grading several java classes at once. It reads the output of the functions and compares it against the expected output. If the output is equal to the expected output, that function gets a 'pass.' Otherwise, that function fails and can be reviewed manually.

C++

Appointment Manager GitHub
Nathan Reline

While this project is not exceptionally functional at recording events, it served as a great milestone in my early programming days and shows my familiarity with the language. The building blocks of this application consist of my own custom built String, DateTime, and Array classes. Shown here is one of my personal favorite snippets.

void MyString::Insert(const MyString & aMyString, int index)
{
int lengthOfAMyString = aMyString.Length();
int newCapacity = _length + lengthOfAMyString;
_capacity = newCapacity;
char *tempStr = new char[_capacity + 1];
for(int i = 0; i < index; i++)
{
tempStr[i] = _string[i];
}
tempStr[index] = '\0';
strcat(tempStr, aMyString._cstr());
for(int i = 0; i <= _length - index; i++)
{
tempStr[index + lengthOfAMyString + i] = _string[index + i];
}
_length = _capacity;
delete [] _string;
_string = tempStr;
}

NESPi GitHub
Nathan Reline

The Nintendo Entertainment System Pi is a portable Raspberry Pi capable of playing NES games. It runs on ArchLinux due it being lightweight, uses Nestopia to emulate NES games using ROMs, and uses various other hardware for portability. For example, the controller was made using an Arduino Leonardo board I programmed in C.

This website! I also made a script to dynamically retrieve a user's GitHub feed using jQuery.

$.ajax({
type: "GET",
url: "http://cors.io/?u=https://github.com/" + $user + ".atom",
cache: false,
dataType: 'xml',
success: function(result) {
console.log($user + ".atom loaded");
var atom = $("<span />", { html: result.firstChild }).text();
var githubActivity = $("#githubActivity");
githubActivity.html(atom);
var children = githubActivity.children();
githubActivity.html(children);
$('.github a').each(function () {
var href = $(this).attr('href');
if (href) {
href = "https://github.com" + href;
$(this).attr('href', href);
$(this).attr('target', "_blank");
}
});
},
async: true
});
