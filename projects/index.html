Android

Asobouyo! Gist
Role: Front-End Developer

My first Android application, あそぼうよ！ (Let's Play!), is an Android application for studying Japanese through listening to songs and filling in the missing lyrics. The login was routed through the Twitter Digits API for Android and the YouTube Android Player API was used to display the music videos. User credentials, score information, and song data is stored on a remote SQL server which is queried with PHP triggered by HTTP requests from the client.

Team: Benedicta Baidoo

Tag! You're It! GitHub
Role: Sole Developer

Tag! You're it! is an Android application that allows users to play tag, but without any of that coming in contact with other humans nonsense. Players are alerted if others in the same game are nearby. Players then proceed to either avoid being tagged or chase their target. The distance between players is reported to the user but not the direction they are, which keeps the players on their toes.

Project Runner GitLab
Role: Front-End Developer

An Android application developed with C++ through the Android Native Development Kit, Project Runner is a 2D Top-Down, endless running platformer where the user looks down at the player and leads their character through a never ending dungeon. Design inspired by Temple Run and art inspired by The Impossible Game, the player will use swipes to change lanes and directions in order to avoid obstacles to achieve a high score.

Team: Patrick Arbasak, Trevor Berninger

Earthquake Hazards GitHub
Role: Sole Developer

A simple Android application that shows recent Earthquakes from around the world, including information such as coordinates, magnitude, and time, as well as filters to narrow down search results.

Intermediary GitHub
Role: Sole Developer

A mobile application for those who have trouble speaking their own or other languages. Choose any supported locale on Android to say any text you input. Only installed languages are supported. Default language is Japanese due to original concept originating from the need to hear the pronunciation of Chinese characters.

Java

Java Function Tester GitHub
Role: Developer

The Java Function tester was an application created for a High School teacher to help with grading several java classes at once. It reads the output of the functions and compares it against the expected output. If the output is equal to the expected output, that function gets a 'pass.' Otherwise, that function fails and can be reviewed manually.

Team: Justin Fern, Kyle Huntsman

Perspective Rectifier GitHub
Role: Developer

A Java application that uses linear algebra to change the perspective of an image. The user interface was developed in JavaFX Scene Builder 2.

Team: Kyle Huntsman, Brandon Olson

Pokéclient GitHub: Old GitHub
Role: Front-End Developer

Pokéclient is a game master/manager program for Pokémon Tabletop United. Pokémon Tabletop United, or PTU for short, is a tabletop RPG system designed for running games in the world of Pokémon. Production began with the intent to release the application with a JavaFX user interface, but the project is now being rehauled with a web-based frontend using the Bootstrap framework and a Java backend.

Team: Justin Fern, Kyle Huntsman, Brandon Olson

C++

Appointment Manager GitHub
Role: Sole Developer

This project explored the fundamentals of C++ by avoiding the use of array and string from the standard template library and ctime to create a simple appointment manager that reads and writes from a simple text file in the root directory of the project.

NESPi GitHub
Role: Sole Developer

The Nintendo Entertainment System Pi is a portable Raspberry Pi capable of playing NES games. It runs on ArchLinux due it being lightweight, uses Nestopia to emulate NES games using ROMs, and uses various other hardware for portability. For example, the controller was made using an Arduino Leonardo board programmed in C.
