Tag! You're It! GitHub
Role: Sole Developer

Tag! You're it! is an Android application that allows users to play tag but without having to physically touch one another. Players will be alerted if others in the same game are nearby. Players will then proceed to either avoid being tagged or chase their target. The distance between players will be reported to the user but not the direction they are, which keeps the players on their toes. The minimum distance at which players need to be will be customizable, as well as the number of players who can be 'it' or start out as 'it'.

Java Function Tester GitHub
Role: Developer

The Java Function tester was an application created for a High School teacher to help with grading several java classes at once. It reads the output of the functions and compares it against the expected output. If the output is equal to the expected output, that function gets a 'pass.' Otherwise, that function fails and can be reviewed manually.

Team: Justin Fern, Kyle Huntsman

Appointment Manager GitHub
Role: Sole Developer

While this project is not exceptionally functional at recording events, it served as a great milestone in his early programming days and shows familiarity with the language. The building blocks of this application consist of my a custom built String, DateTime, and Array classes. Shown here is a personal favorite snippets.

void MyString::Insert(const MyString & aMyString, int index)
{
int lengthOfAMyString = aMyString.Length();
int newCapacity = _length + lengthOfAMyString;
_capacity = newCapacity;
char *tempStr = new char[_capacity + 1];
for(int i = 0; i < index; i++)
{
tempStr[i] = _string[i];
}
tempStr[index] = '\0';
strcat(tempStr, aMyString._cstr());
for(int i = 0; i <= _length - index; i++)
{
tempStr[index + lengthOfAMyString + i] = _string[index + i];
}
_length = _capacity;
delete [] _string;
_string = tempStr;
}

NESPi GitHub

Reline.github.io GitHub
Role: Sole Developer

This website! Here is also a script written during production to dynamically retrieve a user's GitHub feed using jQuery.

$.ajax({
type: "GET",
url: "http://cors.io/?u=https://github.com/" + $user + ".atom",
cache: false,
dataType: 'xml',
success: function(result) {
console.log($user + ".atom loaded");
var atom = $("<span />", { html: result.firstChild }).text();
var githubActivity = $("#githubActivity");
githubActivity.html(atom);
var children = githubActivity.children();
githubActivity.html(children);
$('.github a').each(function () {
var href = $(this).attr('href');
if (href) {
href = "https://github.com" + href;
$(this).attr('href', href);
$(this).attr('target', "_blank");
}
});
},
async: true
});
